# /start — Initialisierungs-Ritual

// turbo-all

Führe **alle** Schritte durch, wenn eine neue Session beginnt.

---

## 1. Kontext laden
Lese den aktuellen Stand (STATE.md, ROADMAP.md) für das Alpine Digital Projekt (speziell Time to Jump Dolomites).

## 2. Überblick verschaffen
Mache dir einen Überblick über ausstehende Roadmap-Ziele und offene Blocker.
Zeige dem User eine extrem kurze Zusammenfassung (2-3 Sätze):
1. Wo stehen wir aktuell (STATE)?
2. Was ist die oberste Priorität laut ROADMAP.md?
3. Frage: "Sollen wir direkt mit [Priorität 1] starten oder gibt es heute etwas Dringenderes?"
