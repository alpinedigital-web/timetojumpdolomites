Einschränkungen
⚠️ Was die Website NICHT kann

Die folgenden Bereiche sind visuell/designtechnisch vorhanden, aber ohne echte Funktion. Für diese Features wäre eine Datenbank und ein Backend notwendig.

🛒

Shop-Bereich

● Platzhalter

Aktuell: Platzhalter-Sektion mit „Coming Soon"-Meldung.

Was fehlt: Produktdatenbank, Warenkorb-Logik, Zahlungsabwicklung (z.B. Stripe), Bestellverwaltung, Liefer-/Versandstatus.

Voraussetzung: Backend + Datenbank + Zahlungsgateway

📅

Eventkalender

● Platzhalter

Aktuell: Platzhalter-Sektion mit „Keine Events"-Meldung.

Was fehlt: Event-Datenbank mit Datum, Titel, Beschreibung, Bildern. Admin-Oberfläche zum Erstellen, Bearbeiten und Löschen von Events.

Voraussetzung: Backend + Datenbank + Admin-UI

📝

Blog

● Platzhalter

Aktuell: Blog-Seite (blog.html) vorhanden, aber ohne Inhalte.

Was fehlt: Artikel-Datenbank, Rich-Text-Editor für Blogposts, Bilder-Upload und -Hosting, Kategorien/Tags.

Voraussetzung: Backend + Datenbank + CMS-Editor

⚙️

Admin-Panel / CMS

● Verfügbar als Upgrade

Ein Admin-Panel ist bereits vorbereitet und kann bei Bedarf aktiviert werden. Damit können alle Texte (in allen 9 Sprachen), Bilder und Links direkt über eine komfortable Oberfläche bearbeitet werden — mit Live-Vorschau, Suchfunktion und Export/Import.

Voraussetzung für Aktivierung: Backend-Server mit Datenbank und sicherer Authentifizierung, damit Änderungen für alle Besucher sichtbar werden.

Voraussetzung: Backend + Datenbank + Authentifizierungssystem

Zusammenfassung

Die Website ist als Landingpage vollständig einsatzbereit. Kontaktformular, Mehrsprachigkeit (9 Sprachen), responsives Design, Sicherheitsmaßnahmen und alle Informationsseiten funktionieren einwandfrei.

Falls in Zukunft Shop, Eventkalender oder Blog gewünscht werden, kann ein Backend nachträglich integriert werden. Der bestehende Code ist so aufgebaut, dass eine spätere Erweiterung problemlos möglich ist.

Eine eigene Domain (z.B. timetojumpdolomites.com) kann jederzeit mit dem aktuellen Hosting verbunden werden.
